## Official summary

- The 32.1 line introduced a major Audio Mixer overhaul, so older tutorials and workflow screenshots may no longer match the current interface.
- WebRTC support was added in the same release cycle.
- OBS's official download page currently lists 32.1.1 as the active release for Windows, macOS, and Linux.
