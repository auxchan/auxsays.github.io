## Key changes

- OBS Studio 32.1 introduced a large Audio Mixer overhaul.
- The 32.1 line also added WebRTC support.
- OBS's official download page lists 32.1.1 as the active release for Windows, macOS, and Linux.

## Official breakdown

- Audio Mixer behavior and layout changed in the 32.1 line, so older tutorials may not match the current interface.
- WebRTC support was added in the 32.1 release cycle.
- Version 32.1.1 is the current maintained release in that line.
