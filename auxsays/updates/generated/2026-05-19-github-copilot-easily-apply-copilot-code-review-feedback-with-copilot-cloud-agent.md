  \ clicking Implement suggestion would generate a comment tagging @Copilot on your behalf to open a new pull request with\

  \ request or open a new pull request targeting your branch.\n Select the model you want Copilot to use when implementing\
  \ the changes.\n Add optional additional instructions to guide the changes.\n\n\n\n\n Ask Copilot to address multiple comments\
  \ at once\n With the new Fix batch with Copilot button in Copilot’s Pull Request Overview comment, you can ask Copilot cloud\

  \ helps you act on multiple comments more efficiently instead of handling each one individually.\n\n\n\n\n\n Join the discussion\
